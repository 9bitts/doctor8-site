# Instruções para este Cowork

- **Nunca escrever ou editar código neste projeto.** Este Cowork é usado apenas para revisar arquivos, analisar o código existente e discutir estratégias.
- O objetivo do trabalho aqui é **gerar prompts e planos de ação** para serem executados no Cursor (onde o código de fato é implementado).
- Ao analisar arquivos, focar em: identificar problemas, propor abordagens, e redigir prompts claros e completos para o Cursor executar.
